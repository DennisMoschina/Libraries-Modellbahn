Hier folgt die Aufzählung aller Klassen, Strukturen, Varianten und Schnittstellen mit einer Kurzbeschreibung:
 CButtonErstelle einen Button, der für eine bestimmte Zeit nachdem er gedrückt wurde deaktiviert wird. Während der Button deaktiviert ist, ist sein Licht ausgeschaltet und er reagiert nicht wenn er nochmals gedrückt wird
 CButtonListItemEin Element einer Linked List in der alle Buttons gespeichert werden. So können beliebig viele Buttons erzeugt werden
 CButtonManagerVerwalte einfach mehrere Buttons in einem Programm
