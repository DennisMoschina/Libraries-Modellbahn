\doxysubsection{Ausführliche Beschreibung}
Erstelle einen \doxyref{Button}{S.}{class_button}, der für eine bestimmte Zeit nachdem er gedrückt wurde deaktiviert wird. Während der \doxyref{Button}{S.}{class_button} deaktiviert ist, ist sein Licht ausgeschaltet und er reagiert nicht wenn er nochmals gedrückt wird. 

Alle Instanzen müssen vor oder im setup erzeugt werden. In der loop muss \doxyref{handle\+Button()}{S.}{class_button_a9d285e131ce246682d1dc2b0f9e5f3b7} ausgeführt werden. Bei der Verwendung von mehreren Buttons wird die verwendung von \doxyref{Button\+Manager}{S.}{class_button_manager} empfohlen. 

Definiert in Zeile 131 der Datei Button.\+h.
