\doxysubsection{Ausführliche Beschreibung}
Erstelle einen \mbox{\hyperlink{class_button}{Button}}, der für eine bestimmte Zeit nachdem er gedrückt wurde deaktiviert wird. Während der \mbox{\hyperlink{class_button}{Button}} deaktiviert ist, ist sein Licht ausgeschaltet und er reagiert nicht wenn er nochmals gedrückt wird. 

Alle Instanzen müssen vor oder im setup erzeugt werden. In der loop muss \mbox{\hyperlink{class_button_a9d285e131ce246682d1dc2b0f9e5f3b7}{handle\+Button()}} ausgeführt werden. Bei der Verwendung von mehreren Buttons wird die Verwendung von \mbox{\hyperlink{class_button_manager}{Button\+Manager}} empfohlen. \begin{Desc}
