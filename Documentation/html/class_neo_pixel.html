Ausführliche Beschreibung

Füge einfach neue LED strips zum Projekt hinzu und steuere diese mit den grundlegensten Funktionen an. Es können nur WS2812B LEDs verwendet werden.

Diese Klasse implementiert nur die grundlegensten Funktionen. Für alles aufwendigere wird empfohlen die FastLED Bibliothek direkt zu verwenden. Derzeit können nur die Pins 2 bis 13 verwendet werden!

Vorbedingung
Um die Funktionen von NeoPixel nutzen zu können, muss zuerst ein Objekt von NeoPixel erzeugt werden.
NeoPixel myLED = NeoPixel(numberOfPixels, pin);

Ändere die Farbe aller LEDs in einem bestimmten Intervall.

Parameter
colorDie Farbe die die LEDs bekommen sollen als Typ CRGB. Die Farbe kann als Hexcode (0xFF0000 für Rot), als CRGB(255,0,0) für Rot, oder als CRGB::Red angegeben werden.
startStart des Intervalls in dem die LEDs die Farbe bekommen sollen. Der Standardwert ist 0
endEnde des Intervalls in dem die LEDs die Farbe bekommen sollen. Der Standardwert ist die letze LED. Wird dieser Wert angegeben, muss auch start angegeben werden.

Werden keine Werte für start und end angegeben, werden hierfür die Standardwerte verwendet.

Diese Methode macht das gleiche wie NeoPixel::fill(int r, int g, int b, int start = 0, int end = -1) und unterscheidet sich von ihr nur in ihren Parametern.

◆ flash()

void NeoPixel::flash ( int  duration = 1)

Simuliere einen Blitz für eine bestimmte Zeit.

Parameter
durationdie Zeit, die der Blitz dauern soll in Sekunden. Der Standardwert ist eine Sekunde.

Der Blitz kann sowohl für Gewitter, als auch für Explosionen eingesetzt werden. Wird kein Wert für duration angegeben, wird der Standardwert 1 verwendet.

Ändere die Helligkeit einer LED oder aller LEDs gleichzeitig.

Parameter
brightnessDie zu setzende Helligkeit. Liegt im Intervall [0, 255]. Der Standardwert ist das Maximum von 255
pixelDie Nummer der LED, die ihre Helligkeit ändern soll. Der Standardwert sind alle LEDs gleichzeitig

Die Farbe der LED(s) bleibt dabei erhalten. Es kann sogar von einer dunklen Farbe, wie CRGB(10, 10, 10) die Helligkeit erhöht werden.
