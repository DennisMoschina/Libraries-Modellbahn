\doxysubsection{Ausführliche Beschreibung}
Füge einfach neue LED strips zum Projekt hinzu und steuere diese mit den grundlegensten Funktionen an. 

Diese Klasse implementiert nur die grundlegensten Funktionen. Für alles aufwendigeres wird empfohlen die Fast\+LED Bibliothek direkt zu verwenden. Derzeit können nur die Pins 2 bis 13 verwendet werden!
