Ausführliche Beschreibung

Der Button wird für eine bestimmte Zeit deaktiviert, nachdem er gedrückt wurde. Um dies zu verdeutlichen, wird das (oft im Button eingebaute) Licht deaktiviert und wieder deaktiviert, sobald diese Zeit abgelaufen ist.

Der Button ist mit dem internen PULLUP-Widerstand angeschlossen, das heißt, dass der Button nur zwischen GND und dem Pin ohne einen externen Widerstand angeschlossen wird.

Zu beachten
Alle Instanzen müssen vor oder im setup erzeugt werdenund die Methode Button::handleButton() muss regelmäßig in kurzen Zeitabständen aufgerufen werden um jeden Druck auf den Button zu registrieren. Die einfachste Möglichkeit hierfür ist
void loop() {
myButton.handleButton();
delay(10);
}
Bei der Verwendung von mehreren Buttons wird die Verwendung von ButtonManager empfohlen.
Achtung
Das Überprüfen des Buttons sollte nicht in einem Timer geschehen, da es so zu Problemen mit delay() und einigen anderen Funktionen kommen kann.
Vorbedingung
Um die Funktionen von Button nutzen zu können, muss zuerst ein Objekt von Button erzeugt werden.
Buttton myButton = Button(buttonPin, lightPin, delayTime);

◆ handleButton()

void Button::handleButton ( )

Überprüfe, ob der Button gedrückt wurde oder ob er wieder aktiviert werden kann.

Wurde der Button gedrückt und ist aktiv, wird die durch setCallback() festgelegte Funktion ausgeführt.

Zu beachten
Diese Methode muss regelmäßig in kurzen Zeitabständen aufgerufen werden um jeden Druck auf den Button zu registrieren. Die einfachste Möglichkeit hierfür ist:
void loop() {
myButton.handleButton();
delay(10);
}
Achtung
Das Überprüfen des Buttons sollte nicht in einem Timer geschehen, da es so zu Problemen mit delay() und einigen anderen Funktionen kommen kann.
Beispiele
ButtonExample.cpp.
