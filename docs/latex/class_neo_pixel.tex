\doxysubsection{Ausführliche Beschreibung}
Füge einfach neue LED strips zum Projekt hinzu und steuere diese mit den grundlegensten Funktionen an. Es können nur WS2812B LEDs verwendet werden. 

Diese Klasse implementiert nur die grundlegensten Funktionen. Für alles aufwendigere wird empfohlen die \href{http://fastled.io}{\texttt{ Fast\+LED Bibliothek}} direkt zu verwenden. Derzeit können nur die Pins 2 bis 13 verwendet werden!

\begin{DoxyPrecond}{Vorbedingung}
Um die Funktionen von \mbox{\hyperlink{class_neo_pixel}{Neo\+Pixel}} nutzen zu können, muss zuerst ein Objekt von \mbox{\hyperlink{class_neo_pixel}{Neo\+Pixel}} erzeugt werden. 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{class_neo_pixel}{NeoPixel}} myLED = \mbox{\hyperlink{class_neo_pixel_a6ad2ee6bce11578331c17cdc9c6f9122}{NeoPixel}}(numberOfPixels, pin); }

\end{DoxyCode}
 {\ttfamily number\+Of\+Pixels} und {\ttfamily pin} müssen dabei vorher angegebn werden oder direkt durch eine Zahl ersetzt werden. 
\end{DoxyPrecond}
